INTRODUCTION
------------

The pfe_cache_purge module provides a way for communicating to 
the dashboard for cache purging for workbench user's.

REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install the pfe_cache_purge module as you would normally install a contributed
   Drupal module. Visit https://www.drupal.org/node/1897420 for further
   information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the module.
